Reddit

Auto-Publisher

Summary

Scrape's top crypto currency news sites and auto-publishes links to r/CryptoCurrency

Github Repo

(Private)

Inspiration

Reddit introduced a reddit based Crypto currency called moons. These moons are rewarded to users who get positive karma on the r/CryptoCurrency subreddit. I thought a clever way to farm upvotes while not spamming would be to share links to breaking news faster than anyone on the subreddit.

Technologies

I used Reddit's developer API, Python and Crontabs to automatically run python scripts
